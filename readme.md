## 🚀 加速机制

### 工作原理

每次酒桶完成**一轮完整酿造**（达到满级 7 级），`酿造次数` 永久 +1。

`酿造次数` 影响当前酒桶的**速度加成**：

- **第 1 次**：+5%
- **第 2 次**：+6.25%
- **第 3 次**：+7.5%
- **第 4 次**：+8.75%
- **第 5 次及以上**：+10%（或配置的最大值）

### 速度公式
速度加成 = 5% + (有效次数 - 1) × (最大速度 - 5%) / (最大次数 - 1)

其中：
- `有效次数` = min(实际次数, `maxBrewCountForSpeed`)
- `最大速度` = `maxSpeedPercent`（默认 10%）
- `最大次数` = `maxBrewCountForSpeed`（默认 5）

### 加速效果

速度加成直接作用于**每个酿造阶段**：
阶段时间 = 原始阶段时间 × (1 - 速度加成)

| 次数 | 速度加成 | 酿造总时间（原始 30秒） |
|------|---------|----------------------|
| 0 | 0% | 30.0 秒 |
| 1 | 5% | 28.5 秒 |
| 2 | 6.25% | 28.1 秒 |
| 3 | 7.5% | 27.75 秒 |
| 4 | 8.75% | 27.4 秒 |
| 5+ | 10% | 27.0 秒 |

## 💥 损坏机制

### 工作原理

每次**完整酿造完成**时，酒桶有概率损坏。
损坏概率 = 5% + 酿造次数 × 5%
最高上限：55%

| 次数 | 损坏概率 |
|------|----------|
| 0 | 5% |
| 1 | 10% |
| 2 | 15% |
| 3 | 20% |
| 4 | 25% |
| 5 | 30% |
| 10 | 55% |
| 10+ | 55%（最高） |

### 损坏效果

- 💥 **TNT 爆炸特效**（无伤害）
- 🔊 **僵尸破门声**
- 🎆 **大量爆炸粒子 + 火焰粒子**
- 🗑️ **移除 3×3×3 范围内的所有方块**（包括装饰方块）

## ⚙️ 配置说明

配置文件位置：`./config/kaletaold-common.toml`

```toml
[brew_speed]
    # 达到最大速度加成所需的酿酒次数（默认5次，范围1~100）
    maxBrewCountForSpeed = 5
    # 最大速度加成百分比（默认10%，范围1~100）
    maxSpeedPercent = 10
配置项	说明	默认值
maxBrewCountForSpeed	达到最大速度所需的次数	5
maxSpeedPercent	最大速度加成百分比	10
##🛠️ 调试命令
需要 OP 权限（2 级）。

命令	说明
/brewdebug finish	立即完成当前酒桶的酿造（满级）
/brewdebug setlevel <0-10>	设置酒桶的酿造等级
/brewdebug addcount	酿造次数 +1
/brewdebug setcount <0-100>	设置酿造次数
/brewdebug damage	💥 摧毁酒桶（爆炸特效）
/brewdebug info	查看酒桶当前状态
/brewdebug info 输出示例
=== 酿酒桶信息 ===
当前酿造等级: 7 / 7
已完成的完整酿造次数: 5
当前速度加成: +10%
下次损坏概率: 30%
